# 🧠 Breast Cancer Prediction System

This project predicts whether a tumor is **malignant (cancerous)** or **benign (non-cancerous)** using machine learning.

---

## 🚀 Live Demo
🌐 https://breast-cancer-predictor-koushik.streamlit.app

---

## 📊 Dataset
- Breast Cancer Wisconsin Dataset (UCI ML Repository)

---

## ⚙️ Features Used
- radius_worst
- perimeter_worst
- area_worst
- concave_points_worst
- radius_mean
- texture_mean
- concavity_mean

---

## 🤖 Models Used
- Random Forest (selected model)
- XGBoost
- Gradient Boosting

---

## 📈 Performance
- Accuracy: ~95%
- Strong recall for malignant cases

---

## 🧩 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

---

pip install -r requirements.txt
